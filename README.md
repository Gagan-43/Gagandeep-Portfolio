# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript**. This portfolio showcases my skills, projects, and contact information with a clean and user-friendly design.

## ✨ Features

- Responsive design for all devices
- Clean and modern UI
- Smooth navigation
- About Me section
- Skills showcase
- Projects section
- Contact section
- Easy to customize

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
